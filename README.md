# Rime星空键道

## Description
星空键道6词库，更新同步于[Rime键道主项目][204]。

## 注意事项
本项目不含 ``xkjd6.user.dict.yaml`` / ``default.yaml`` (或 ``default.custom.yaml``)，故仅适合更新之用.
烦请第一次安装的用户，移至[键道6主项目][204]下载。
